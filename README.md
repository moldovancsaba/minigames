# Kids Swipe Game

Small Next.js swipe game for kids built for mobile-first play and Vercel deployment.

## Features

- Start screen with a simple `Start` button
- Swipe round using the image cards from `public/images`
- Vote round for ranking shortlisted cards head-to-head
- Result screen with internal scrolling and a `Restart` button
- Portrait and landscape backgrounds with responsive layout sizing
- PWA-friendly viewport settings with no page scroll and no zoom

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
npm run build
npm run start
```

## Deploy

Deploy the repository to Vercel as a standard Next.js app.
